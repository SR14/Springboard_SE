How the Web Works Exercise Solutions

Part One: Solidify Terminology

What is HTTP? HTTP is a type of communication convention between computers.

What is a URL? URLS contain information that is broken up and used to retrieve information from servers.

What is DNS? Naming system to turn hostnames into IP Adresses.

What is a query string Additional information passed on during a request to specify parameters within the requested page.

What are two HTTP verbs and how are they different? GET and POST are two types of HTTP verbs/methods. The GET verb does not change any of the data at the server; examples include links. The POST verb changes the data at the server; examples include sending an email.

What is an HTTP request? An HTTP request specifices the place where you want to get your information from.

What is an HTTP response? An HTTP response returns the information found at the location the HTTP request indicated.

What is an HTTP header? Examples fo request and response headers. An HTTP header contains specific information about the request and response currently being made. A common header for a request is the Host: header which indicates the hostname requested. A common response header is Content-Type: which indicates the type of content the response will return.

What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser? First the URL will get broken down into its main components, which are the protocol (in this case HTTP), hostname (somesite.com), port (default 80), resources (some/page.html), and query (no additional parameters passed in this case). After the URL gets broken down the components get sent to a server which may or may not interact with other servers to finally respond with the content found at the URL sepcified.

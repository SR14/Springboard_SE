Conceptual Questions: Box Model

  • What is the box model?

    • The box model defines the spacing of each piece of content; it includes the sizing of the content, padding, border and margin.

  • What is the difference between margin and padding?

    • Padding defines the space between the content box and the border box. On the other hand, margin defines the space between the border and other content.

  • What is collapsing margin?

    • Collapsing margin is when two margins get combined into one to preserve uniformity. If looked at individually, both content borders will have the margin specified, but the total margin will be that of only the largest margin.

  • What is the purpose of the box-sizing CSS property?

    • Box sizing lets you define whether the size set for the element describes only the content box or everything including the border.

  • What does the outline property do?

    • The outline property lets you create a outline outside of the border lines. This way the outline does not take up the content's space inside of the border.

  • How do you add a background image in CSS?

    • You add a background image by using the property background-image: url(""); and setting the value to the location of the image desired.

  • Given a div with a width of 200px, margin of 20px and padding of 20px, what is the calculated width by the browser?

    • Since margin is not factored into the width calculation of an element, only the padding will be considered. The 200px element will get padding of 20px on the left and 20px on the right for a total width of 240px.

  • Given a div with a height of 100px, border of 10px and padding of 20px, what is the calculated height by the browser?

    • Margin is not factored in to the dimensions of an element but the border is. Therefore, the total height of the div element will be 100px + 20px(padding top) + 20px(padding bottom) + 10px(border top) + 10px(border bottom), for a total height of 160px.

  • What does margin:auto do?

    • The property:value margin:auto can be used to center an element along the width of a page, and it dynamically centers the element as the browser window is shrunk or enlarged.
